AA MarketContext — overlay en precio
VWAP de sesión — línea blanca gruesa
Precio promedio ponderado por volumen desde el inicio de la sesión activa. Actúa como el nivel de referencia institucional. El precio tiende a gravitarse hacia él durante el día. reset al cambiar bloque Asia / London / NY
SD1 Upper / Lower — líneas azules discontinuas
Desviación estándar ×1.0 del VWAP (canal azul). El 68% del volumen se negocia dentro de este canal. Precio entre ±SD1 = mercado en equilibrio — las señales deben ser más precisas. Precio rozando SD1 desde dentro = posible rebote hacia el VWAP.
SD2 Upper / Lower — líneas rojas discontinuas
Desviación estándar ×2.0. Fuera de este canal = zona de extremo estadístico. Precio en SD2 Upper = zona premium extrema, probabilidad alta de reversión. Precio en SD2 Lower = zona discount extremo, zona de búsqueda de longs institucionales.
Fondo amarillo muy tenue — sesión Asia
Peso de confianza 40% — el mercado tiene poca liquidez, rangos pequeños, falsas rupturas frecuentes. El código aplica alpha=7 (casi imperceptible). No operar señales agresivas en esta zona.
Fondo azul tenue — sesión NY / Overnight
NY AFT (70%) y NY PRE (55%) reciben fondo azul con alpha=9. La etiqueta de texto "OVERNIGHT" aparece sobre el precio al cambiar de bloque.
Fondo azul más intenso — KillZone activa
London KZ (03–05h ET, peso 90%) y NY KZ (09:30–11:30h ET, peso 95%) reciben alpha=14 más visible. Dentro de estos rangos las señales SMC tienen máxima prioridad. Captura es cuando visible en el screenshot (sesión Asia activa, no en KZ).
Fondo rojo intenso — noticia de alto impacto
Máxima prioridad. Cuando NewsImpact=3 (HIGH) y el tiempo actual está dentro de ±15 minutos del evento, el fondo se torna rojo con alpha=30 y se dispara alerta sonora. TradingAllowed=0.0 — el RiskEngine bloqueará cualquier operación.
AA SmartMoneyEngine — overlay en precio, sin BackBrush
Línea azul sólida + texto "BOS ▲" — Break of Structure alcista
El código detecta que el cierre rompió sobre el último swing high con al menos SwingStrength=5 pivotes de separación. Se traza una línea horizontal azul hacia atrás (hasta 20 barras) y se escribe "BOS ▲" junto al nivel. StructureBias = +1 — las barras del fondo cambian a sesgo alcista.
Línea roja sólida + texto "BOS ▼" — Break of Structure bajista
Cierre bajo el último swing low. StructureBias = -1. Indica que la estructura bajista está confirmada y las entradas en dirección contraria son de alto riesgo.
Línea naranja discontinua + "CHoCH" — Change of Character
Primera señal de reversión: el precio viene de estructura bajista y rompe el swing high previo (o viceversa). NO es confirmación — es alerta temprana. Si el siguiente BOS confirma la nueva dirección, el CHoCH fue la entrada correcta.
Rectángulo verde translúcido — FVG alcista
Gap de tres velas donde Low[0] > High[2] y el tamaño supera FvgMinSizeATR × ATR (default 0.25 ATR). El rectángulo cubre el gap. El precio tenderá a regresar a rellenarlo antes de continuar. se oscurece / desaparece al ser mitigado
Rectángulo rojo translúcido — FVG bajista
Gap donde High[0] < Low[2]. Zona de oferta desequilibrada — el precio tenderá a volver a llenarla desde abajo como resistencia.
Caja azul más intensa + "OB 3★" — Order Block alcista (score ≥3)
Última vela bajista antes de un impulso alcista superior a 1.8 × ATR. La opacidad del relleno crece con el score (40 + score×10 de alpha). Solo muestra la etiqueta de puntuación si score ≥ 3. Se invalida (desaparece) si el precio cierra dentro del bloque.
Línea punteada verde — Buy-side liquidity (Equal Lows)
Nivel donde el Low fue tocado ≥2 veces dentro de la tolerancia LiqEqualPct × ATR. Representa stops de compradores acumulados — los algoritmos institucionales tienden a bajar el precio a barrer esta liquidez antes de subir. Cuando el precio lo perfora y recupera = entrada.
Línea punteada roja — Sell-side liquidity (Equal Highs)
Stops de vendedores acumulados sobre los igual-highs. El precio sube a buscarlos antes de revertir. Nivel de target para operaciones long activas.
AA WyckoffVolume — overlay en precio
Línea magenta — POC (Point of Control)
Precio donde se negoció el mayor volumen acumulado del día. Actúa como el "centro de gravedad" — el precio tiende a gravitarse hacia el POC durante el día. El acumulador es incremental (O(n), corregido vs v1): solo suma la barra actual al diccionario tick→volumen, nunca recorre el historial completo.
Líneas azules discontinuas — VAH y VAL (Value Area)
El 70% del volumen del día se negoció entre VAH y VAL. Si el precio está dentro del VA = mercado en equilibrio, esperar ruptura. Si el precio está fuera y regresa al VA = posible compresión. Ruptura del VAH con volumen = señal de continuación alcista.
Líneas verdes punteadas — HVN (High Volume Nodes)
Niveles donde el volumen supera el 150% del promedio. Máximo 5 líneas visibles para no saturar. Actúan como zonas de freno y soporte/resistencia dinámica. El código excluye los HVN dentro de ±0.3 ATR del POC para evitar solapamiento.
Rectángulo amarillo muy tenue — rango Fase B de Wyckoff
Se dibuja cuando se detectan ≥10 barras laterales (precio contenido dentro del rango sin cierre fuera). Delimita el rango de acumulación/distribución. El Spring o UTAD que lo perfora brevemente es la señal de Fase C.

Línea verde — P_Long (probabilidad alcista)
El modelo XGBoost devuelve la probabilidad calibrada de que el precio suba en las próximas 12 barras M5. En modo OFFLINE usa el último valor válido o el fallback 34%. El umbral operativo es MinConfidence = 0.55.
Línea roja — P_Short (probabilidad bajista)
Probabilidad de movimiento bajista. Cuando P_Short > 0.65 y la estructura SMC es bajista y el score >60, se genera señal SHORT. La suma P_Long + P_Short + P_Neutral debe ser ≈1.0 — si no, la respuesta del pipe se descarta.
Línea gris — P_Neutral
Probabilidad de que el precio permanezca dentro del rango sin movimiento tendencial. Cuando P_Neutral domina, el score cae y el sistema emite "SIN SENAL".
Línea blanca — ConsolidatedScore (0–100)
Score compuesto: ML hasta 30pts + sesión hasta 20pts + EMAs hasta 15pts + RSI hasta 10pts + estructura SMC hasta 15pts + Wyckoff hasta 10pts. El panel del screenshot muestra SCORE: 0/100 porque el modelo ML está offline y sesión Asia tiene peso bajo (40%).

Línea roja discontinua — Stop Loss dinámico
Calculado como Close[0] − ATR × 1.5 × StopMultiplier. El StopMultiplier viene de AA_MarketContext: si volatilidad es normal =1.0, si es elevada =1.5, si es extrema el sistema bloquea la entrada directamente. El stop se redibuja en cada barra — siempre refleja la distancia real de riesgo.
Línea verde discontinua — Target 1 (1.5R)
Precio al cual cerrar el 50% de la posición. A ese nivel el código activa el método RegisterExit parcial y mueve el stop a breakeven +0.25 ATR. En el screenshot: T1 = 66,313.
Línea azul discontinua — Target 2 (2.5R)
Precio para cerrar el 30% restante. A este nivel el trailing stop se activa si no estaba ya activo. En el screenshot: T2 = 66,469.
Línea naranja sólida — Trailing Stop activo
Aparece solo cuando trailingActive = true, es decir cuando el trade supera TrailActivationR = 1.5. Se mueve en dirección del trade barra a barra. Si el precio lo toca, el código llama a RegisterExit(trailingStopPrice) y se marca "TRAIL EXIT".
